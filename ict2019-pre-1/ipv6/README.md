# VyOSの設定が吹き飛んだ！[IPv6]
## 問題概要
```
社内のラボ環境でVyOSにサーバ、クライアントを接続し、相互にIPv6で通信できる環境を構築していた。
ある日VyOSの再起動を行ったところ、設定ファイルの保存を行っていなかったためVyOSの設定が消えてしまった。
VyOSを以下の要件に従って設定を行いクライアント-サーバ間で通信ができるように復旧し、管理者用アカウントを追加して欲しい。
また、今後同様の事象が発生しないように対策を講じること。
```
